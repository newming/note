      # 分区

      分区设备文件名：

      • /dev/hda1 (IDE硬盘接口)
      • /dev/sda1 (SCSI 硬盘接口，SATA 硬盘接口)

      必须分区

      • / (根分区)
      • swap 分区 (交换分区，内存2倍，不超过2GB)

      推荐分区

      • /boot (启动分区， 200MB)
