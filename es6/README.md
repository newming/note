# es6 部分语法

- let-const: let, const的用法及块级作用域
- Destructuring_assignment: 解构赋值操作
- String: 字符串拓展
- RegExp: 正则拓展
- Number: 数值拓展
- Array: 数组拓展
- Function: 函数拓展
- Object: 对象拓展
- Symbol: 新增原始数据类型
- Set,Map: Set, Map, WeakSet, WeakMap
- Proxy: 代理
- Reflect: Object 属性代理对象
- Promise: Promise 对象
- Iterator: Iterator 和 for...of 循环
- Generator: Generator 函数
