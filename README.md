### 本地安装的说明

- 首先安装全局的 gitbook 命令行 npm install -g gitbook-cli （相当于nvm的作用）
- 然后通过git book 安装真正使用的 gitbook 版本 gitbook --help 查看
- gitbook fetch [version]
- 编译代码的帮助命令 gitbook help
- gitbook serve 生成静态文件
