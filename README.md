### redis

```bash
brew install redis
brew services start redis # redis 服务常驻，开机自启动
redis-server /usr/local/etc/redis.conf # 启动redis服务
redis-cli # 链接 redis
set [key] [val] # redis 设置命令
get [key] # 获取命令
keys * # 获取所有的key
```
